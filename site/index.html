Hello World Skill (Multi-Vendor)

v0.1.0 • Copilot + Claude Code + OpenCode • POC

This is the first published skill demonstrating vendor-neutral AI tool integration. One canonical skill automatically transforms into three vendor-specific distributions for GitHub Copilot, Claude Code, and OpenCode.

📥 Installation Instructions by Vendor:
🔧 GitHub Copilot CLI
Format: plugin.json
Development:
copilot plugin install CG-Global-AIforSWE/aif-optimisation-utils:dist/copilot/marketplace/skill.atomic.hello-world
Release (Stable):
copilot plugin install github:CG-Global-AIforSWE/aif-optimisation-utils:dist/copilot/marketplace/skill.atomic.hello-world
🤖 Claude Code
Format: .claude/skills
Option 1: Local Copy (Fastest):
mkdir -p ~/.claude/skills && cp -r path/to/aif-optimisation-utils/dist/claude/.claude/skills/skill.atomic.hello-world ~/.claude/skills/
(Replace 'path/to' with your actual repo path)
mkdir -p $env:USERPROFILE\.claude\skills; Copy-Item -Recurse C:\path\to\aif-optimisation-utils\dist\claude\.claude\skills\skill.atomic.hello-world $env:USERPROFILE\.claude\skills\
(Replace 'C:\path\to' with your actual repo path)
Option 2: Sparse Clone (Git):
git clone --depth 1 --filter=blob:none --sparse https://github.com/CG-Global-AIforSWE/aif-optimisation-utils.git
cd aif-optimisation-utils && git sparse-checkout set dist/claude
mkdir -p ~/.claude/skills && cp -r dist/claude/.claude/skills/skill.atomic.hello-world ~/.claude/skills/
Release (Stable):

Use the same commands as Development above. Release tag support coming in v0.2.0.

🎯 Plugin Distribution Model:
  • GitHub-Based Distribution: Plugins are installed directly from your GitHub repository, not a centralized marketplace
  • Auto-Install on Plugin Install: The onPluginInstall hook automatically copies skills to .github/skills/
  • Zero Manual Steps: No need to manually copy files—everything happens automatically
  • Vendor-Neutral: Write once, generate for all vendors (Copilot, Claude, OpenCode, etc.)
  • Future-Ready: Same pattern scales to build-tool-generator and other advanced use cases
Key Features:
  • Write governance rules and skills once in canonical format
  • Automatically generate vendor-specific versions
  • Human verification before publishing
  • One source of truth across all AI tools
  • Plugin distribution with automatic skill installation
