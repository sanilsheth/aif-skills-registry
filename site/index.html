✨ Hello World Skill
v0.1.0 • POC (Proof of Concept)

A simple demonstration skill to test skill distribution across multiple AI tools. Perfect for verifying the HTTP-based installation mechanism works in your environment.

✅ What You Need: Just your AI tool (Copilot, Claude, OpenCode) — no GitHub account required!
Choose your AI tool:
🔧 GitHub Copilot CLI
Installation:
bash -c "$(curl -sSL https://sanilsheth.github.io/aif-skills-registry/dist/copilot/marketplace/skill.atomic.hello-world/hooks/install.sh)"

Run in your terminal, then use: copilot -- /hello-world

🤖 Claude Code
Installation:
mkdir -p ~/.claude/skills/skill.atomic.hello-world && curl -sSL https://sanilsheth.github.io/aif-skills-registry/dist/claude/.claude/skills/skill.atomic.hello-world/SKILL.md -o ~/.claude/skills/skill.atomic.hello-world/SKILL.md

Then restart Claude Code and use the skill
